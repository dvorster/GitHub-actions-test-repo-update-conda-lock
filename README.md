# GitHub-actions-test-repo-update-conda-lock
A Test Repository to test creating a GitHub action to update a Conda-lock file

By: Devon Vorster :)

Create a an automated workflow for creating a conda-lock file usign github acitons.

1. Update environment
2. Update environment.yml file
3. Ensure proper documentation of package versions (using script from Sky)
4. Commit and push to github
5. Github actions will create the conda-lock file for linux-64 :)
